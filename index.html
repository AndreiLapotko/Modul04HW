Зачетная работа по модулю №4 "JS.База"

Задача

  • Создайте переменную с именем myName и присвойте ей ваше имя. Затем напишите функцию greet(), которая выводит приветствие с вашим именем
  • Напиши функцию celsiusToFahrenheit , которая принимает температуру в градусах Цельсия и возвращает её эквивалент в градусах Фаренгейта
  • Напишите функцию calculateFallDistance(), которая принимает время падения объекта в секундах и возвращает расстояние, которое объект пролетел за это время, учитывая ускорение свободного падения (g), равную 9.8 м/с².
    Формула для расчета расстояния падения: d = (1/2) * g * t^2
  • Напишите функцию calculateAverage(), которая принимает три числа в качестве аргументов и возвращает их среднее значение
  • Напишите функцию concatStrings(), которая принимает два слова и возвращает строку “Первое слово - « » , второе слово - « » “

Каждое задание должно быть оформлено в качестве модуля и вызываться в одном файле script.js
